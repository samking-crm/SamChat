# SamChat# SamChat 🚀 - WhatsApp Clone MVP

Production-ready real-time chat app built with React + Node.js + Socket.io

## ✨ Features
✅ Real-time messaging  
✅ Message status (sent/delivered/seen)  
✅ Typing indicators  
✅ Online status  
✅ Mobile-first WhatsApp UI  
✅ Dark/Light mode  
✅ Authentication (Email + Google)  
✅ Media uploads (Cloudinary)  
✅ WebRTC calling ready  
✅ Social feed system  
✅ Premium features ready
